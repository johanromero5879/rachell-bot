# RachellBot
A discord bot

## Requirements
- Node: 14.16.1

## Installation
```
git clone https://github.com/johanromero5879/rachell-bot.git
cd rachell-bot
npm install
```

## Settings
Copy or rename `.env.example` to `.env` and fill out the values:
```
PORT=<SERVER_PORT>
PREFIX=<PREFIX>
BOT_TOKEN=<BOT_TOKEN>
COLOR_BOT=DEFAULT
OWNER_ID=<OWNER_ID>
```

After settings you can execute `npm run dev` to start the bot on dev mode
